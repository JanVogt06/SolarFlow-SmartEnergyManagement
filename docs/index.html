Quick Start

In 3 Schritten zum optimierten Eigenverbrauch

1

Installation

Repository klonen und Dependencies installieren

git clone https://github.com/JanVogt06/SolarFlow-SmartEnergyManagement.git cd SolarFlow-SmartEnergyManagement pip install -r requirements.txt
2

Konfiguration

Geräte in devices.json definieren

{
    "name": "Waschmaschine",
    "description": "Waschmaschine Keller",
    "power_consumption": 2000,
    "priority": 3,
    "min_runtime": 90,
    "max_runtime_per_day": 0,
    "switch_on_threshold": 2100,
    "switch_off_threshold": 100,
    "allowed_time_ranges": [
        ["08:00", "20:00"]
    ]
}
3

Starten

Monitor mit IP deines Wechselrichters starten

python main.py --ip 192.168.178.90
