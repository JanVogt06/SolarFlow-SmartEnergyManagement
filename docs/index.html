Das Problem

"Ich verschenke 60% meines Solarstroms ans Netz"

  • Niedrige Einspeisevergütung (8 Cent/kWh)
  • Hohe Strompreise (40 Cent/kWh)
  • Manuelle Gerätesteuerung ineffizient
→

Die Lösung

"Ich nutze 85% selbst dank intelligenter Steuerung"

  • Automatische Gerätesteuerung
  • Prioritätsbasierte Einschaltung
  • Maximaler Eigenverbrauch

Quick Start

In 3 Schritten zu mehr Eigenverbrauch

1

Installation

Clone das Repository und installiere die Abhängigkeiten

git clone https://github.com/JanVogt06/SolarFlow-SmartEnergyManagement.git cd SolarFlow-SmartEnergyManagement pip install -r requirements.txt
2

Konfiguration

Passe die config.yaml an deine Anlage an

{
    "name": "Waschmaschine",
    "description": "Waschmaschine Keller",
    "power_consumption": 2000,
    "priority": 3,
    "min_runtime": 90,
    "max_runtime_per_day": 0,
    "switch_on_threshold": 2100,
    "switch_off_threshold": 100,
    "allowed_time_ranges": [
        ["08:00", "20:00"]
    ]
}
3

Start

Starte den Energy Manager

python solar_monitor.py
